#Proyecto AEM: Simplified Swarm Optimisation (SSO) aplicado en MMKP.

Integrantes: Camilo Ríos, Maximiliano Juárez.
Fecha: Junio 2026.

Para ejecutar el algoritmo en las 15 instancias es necesario tener instalado Python y se ejecuta el archivo sso.py (no requiere librerias adicionales). 

Los resultados se guardan en una subcarpeta de la carpeta 'Resultados' con la fecha de la ejecución. El resultado contiene 2 archivos .png (gráficos de convergencia y % de soluciones factibles) y un resumen en .txt de los logs por instancia. Además, se guarda un archivo .csv con resultados generales de la ejecución con el objetivo de comparar los valores obtenidos para cada instancia.
